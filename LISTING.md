# How to edit listing

1. Open the [ConfigCat Jira Plugin on the Marketplace](https://plugins.jetbrains.com/plugin/26096-configcat-feature-flags/edit) with an admin account
2. To edit the "Description" section, change the README.md (see below)
3. To edit the "Getting Started" click the "Edit section" button

> **Note**
> The current images can be found on the GDrive > Design-Illustration-Graphics > integrations folder.

# Plugin description

The plugin description can be edited in the plugin README.md. The plugin will use text between the
<!-- Plugin description --> tags when building the plugin. 

# Getting Started
Current value: 

<h3>Usage of ConfigCat Feature Flags Plugin</h3>
<p>The ConfigCat Feature Flags Tool Window can be opened by clicking the ConfigCat Feature Flags icon on the side of the IDE main window.</p>
<h3>Authentication</h3>
<ol>
<li>Get your ConfigCat Public API credentials from <a href="https://app.configcat.com/my-account/public-api-credentials">ConfigCat Dashboard/Public Management API credentials</a>.</li>
<li>Open the <code>File</code> &gt;  <code>Settings/Preferences</code> menu.</li>
<li>Search for <code>ConfigCat Feature Flags</code> page.</li>
<li>Fill out the <code>Basic auth user name</code> and <code>Basic auth password</code> inputs.</li>
</ol>
